To Connect to the database before running the program:

1. Click on Services
2. Right Click on Java DB
3. Click on 'Properties'
4. In JAVA DB properties, please point Database Location    to the path to where 'WhatsCooking' was extracted
2. Right Click on Java DB click 'start server'.
